# Honey-Bees-and-Tropilaelaps_DDE

This repository contains the code used to generate the figures for 
the population dynamics of European honey bees and _Tropilaelaps_ mites with age structure,
infestation–infection dynamics, and seasonal forage.

For each figure, the corresponding code is provided in this repository under the relevant file title.

## Contents

- Jupyter notebooks (`.ipynb`) containing the simulations

## Remarks

The file names indicate the figure number or analysis to which they correspond.
